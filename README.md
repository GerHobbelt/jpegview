# JPEGView-Image-Viewer-and-Editor

> Updated Dec 01 2021.

Second life for famous JPEGView - fast and tiny viewer/editor for JPEG, BMP, PNG, WEBP, TGA, GIF and TIFF images with a minimalist GUI and base image processing.
